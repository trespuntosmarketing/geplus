Conectar base de datos
GE+ usa Supabase como base de datos gratuita en la nube. Solo necesitas hacer esto una vez.
⚡ Pasos rápidos (5 minutos):

1. Ir a supabase.com → Crear cuenta gratis
2. Crear nuevo proyecto (anotar la contraseña)
3. Ir a Settings → API
4. Copiar Project URL y anon public key
5. Pegar abajo y hacer clic en Conectar
